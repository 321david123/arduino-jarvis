┌─────────────────────────────────────────────────────────────────────────┐
│                        WIRING CHECKLIST                                 │
└─────────────────────────────────────────────────────────────────────────┘

Before connecting power, verify:

  OLED Display:
  □ VCC → 3.3V (or 5V)
  □ GND → GND
  □ SCL → GPIO22
  □ SDA → GPIO21

  INMP441 Microphone:
  □ VDD → 3.3V
  □ GND → GND
  □ SCK → GPIO14
  □ WS  → GPIO15
  □ SD  → GPIO32
  □ L/R → GND  ⚠️ CRITICAL!

  Final Checks:
  □ No loose connections
  □ No short circuits between pins
  □ All power connections correct
  □ All ground connections in place

┌─────────────────────────────────────────────────────────────────────────┐
│                          SAFETY NOTES                                   │
└─────────────────────────────────────────────────────────────────────────┘

  ⚠️  ALWAYS disconnect power before making wiring changes
  ⚠️  DO NOT exceed 3.3V on any GPIO pin
  ⚠️  Avoid short circuits between VCC and GND
  ⚠️  Use proper USB cable (data + power, not power-only)
  ⚠️  Keep components away from liquids and metal objects


┌─────────────────────────────────────────────────────────────────────────┐
│                    TROUBLESHOOTING QUICK TIPS                           │
└─────────────────────────────────────────────────────────────────────────┘

  Problem                   →   Check
  ═══════════════════════════════════════════════════════════════════
  OLED blank                →   SDA/SCL not swapped, try 5V for VCC
  No audio detected         →   L/R pin connected to GND
  Garbled display           →   I2C address (should be 0x3C)
  No WiFi connection        →   Correct SSID/password, 2.4GHz network
  Upload fails              →   Hold BOOT button, check USB cable
